      Return unique contexts as a DataFrame.

      :param context_infos: Optional context filters. When omitted, returns all unique contexts.
      :type context_infos: list[ContextInfo] | None, default None
      :param with_partition_col: Whether to include the raw partition column alongside the expanded
                                 context keys.
      :type with_partition_col: bool, default False

      :returns: A DataFrame of unique contexts. When ``with_partition_col`` is
                ``True``, the partition column is retained in the result.
      :rtype: pl.DataFrame



   .. py:method:: get_list(context_infos: list[ContextInfo] | None = None, with_partition_col: bool = False) -> list[ContextInfo]

      :type context_infos: list[ContextInfo] | None, default None
      :param with_partition_col: Whether to include the raw partition column in each returned item.
      :type with_partition_col: bool, default False

      :returns: Unique context dictionaries, optionally filtered by the provided

      :rtype: list[ContextInfo]
