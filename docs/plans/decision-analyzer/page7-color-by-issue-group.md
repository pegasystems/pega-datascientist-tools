# Color optionality plot by issue/group (Page 7 — Optionality Analysis)

**Priority:** P2
**Touches:** `python/pdstools/app/decision_analyzer/pages/7_Optionality_Analysis.py`, `plots.py`

Bi-modal patterns in the optionality distribution are hard to explain without knowing which issues/groups drive which part of the distribution.

## Approach

Add a selector to color the optionality distribution by Issue or Group. Helps explain bi-modal patterns where different issues/groups dominate different parts of the distribution.
